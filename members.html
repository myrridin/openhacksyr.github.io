All attendees are welcome to join the members page!

Join our members page by taking the following steps:

  1. Attend an OpenHack!
  2. Fork the website repository on github
  3. Add your info to the member file
  4. Open a Pull Request on Github (See instructions here)

If any of this seems daunting, please bring it up during the next OpenHack, we'd be happy to help.
